# advent-of-code-rescript

> Advent of Code in ReScript

## Getting started

### Installation

```shell script
yarn
```

### Development

```shell script
# Watch
yarn dev

# Build
yarn build
```

### Run

```shell script
# node solution.bs.js
npx nodemon solution.bs.js
```
